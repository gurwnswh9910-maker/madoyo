---

### 1. 🐍 Python Pro (파이썬 전문가 모드)
*   **활용 상황:** `profilecollect.py` 같은 파이썬 스크립트의 성능을 올리거나, 새로운 기능을 추가할 때.
*   **할 수 있는 것:** 
    *   파이썬 최신 문법과 베스트 프랙티스(Best Practice) 적용.
    *   비동기 프로그래밍(`asyncio`)을 통한 수집 속도 최적화.
    *   메모리 관리 및 코드 가독성 극대화.

### 2. 🔍 Systematic Debugging (체계적 디버깅)
*   **활용 상황:** "원인 모를 오류가 발생해요", "갑자기 작동을 안 해요" 같은 상황일 때.
*   **할 수 있는 것:**
    *   단순히 찍어맞추는 식이 아니라, **'현상 분석 -> 가설 설정 -> 검증 -> 근본 원인 제거'**의 4단계 정밀 디버깅 수행.
    *   동일한 문제가 다시 발생하지 않도록 방어 코드(Defense-in-depth) 작성.

### 3. 🛡️ Security Auditor (보안 감사)
*   **활용 상황:** 만든 프로그램을 서버에 올리거나, 보안이 중요한 데이터를 다룰 때.
*   **할 수 있는 것:**
    *   코드 내에 숨겨진 보안 취약점(Secret 유출, 인젝션 공격 등) 탐색.
    *   개인정보나 로그인 정보가 안전하게 처리되는지 검토.

### 4. 📈 Data Scientist / Data Engineer (데이터 전문가)
*   **활용 상황:** 수집한 엑셀 데이터를 분석하거나 시각화하고 싶을 때.
*   **할 수 있는 것:**
    *   Pandas 라이브러리를 활용한 복잡한 데이터 정제(Cleaning).
    *   데이터 간의 상관관계 분석 및 통계 리포트 생성 가이드.

### 💡 어떻게 사용하나요?
사용자님이 특별히 명령을 내리지 않으셔도, 제가 작업을 하면서 관련 스킬이 필요하다고 판단되면 **스스로 해당 스킬의 가이드라인(`SKILL.md`)을 읽고 적용**합니다.
