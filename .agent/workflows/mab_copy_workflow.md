# MAB 카피 최적화 워크플로우

사용자님의 Threads 계정 성과를 분석하여 최적의 마케팅 카피를 생성하고 검증하는 공식 프로세스입니다.

## 1. 데이터 준비
- `dotori.xlsx` 파일에 최신 포스팅 실적(조회수, 첫댓글조회수 등)이 입력되어 있는지 확인합니다.

## 2. 엔진 가동 및 분석
- `mab_copy_system_v3.py`를 실행하여 다음을 수행합니다.
  1. **데이터 로드**: `dotori.xlsx`에서 최신 성과를 읽어 MSS 지수 산출.
  2. **밴딧 전략 선택**: 내 계정에서 최근 가장 잘 터진 전략 3개 선정 (일간 감쇠 반영).
  3. **후보 생성**: 선정된 전략 기반으로 9개의 카피 후보 생성.
  4. **AI 스코어링**: 스코어러 sub-agent가 유사도와 성과를 분석하여 Top 3 선별.

## 3. 결과 활용
- 출력된 Top 3 카피 중 하나를 골라 포스팅합니다. 
- 스코어러가 제시한 **'평가 사유'**를 통해 어떤 요소가 내 계정에서 먹히는지 인사이트를 얻습니다.

## 4. 피드백 루프 완성
- 포스팅 후 성과 데이터가 쌓이면 다시 1번으로 돌아가 반복합니다. 반복할수록 엔진은 사용자님의 말투와 성공 공식에 더 날카롭게 최적화됩니다.
