# Account-Specific MAB 피드백 루프 시스템 완료 (v3.2)

사용자님의 피드백을 반영하여, **내 계정의 성과를 최우선으로 학습**하는 진정한 의미의 밴딧 피드백 루프 시스템을 구축했습니다. 특히 하루 10건 이상의 대량 포스팅 워크플로우를 반영하여 '일간 감쇠(Daily Decay)' 로직을 적용했습니다.

---

## 핵심 로직: 내 계정 중심의 학습 (Account Priority)

1.  **사용자 데이터 가중치 부여 (1.5 vs 0.5)**:
    - 외부 데이터는 기초 지선으로 활용하고, 사용자님의 데이터는 **3배**의 신뢰 가중치를 가집니다. 

2.  **로직 우선 평가 (Logic-First Scoring)**:
    - 톤앤매너(말투)보다 **성공하는 구조(Hook/Structure)**를 최우선으로 평가합니다. 말투가 조금 달라도 외부 대폭발 로직이라면 과감하게 수용합니다.

3. **성과 중심 스코어링 (Performance-Focus Scoring)**:
    - 9개의 후보 중 '말투(Tone)'가 아닌 **'전략적 구조'와 '성과 예측치(MSS)'**를 최우선으로 분석합니다. 내 말투와 다르더라도 성과가 압도적이라면 과감하게 추천합니다.

4. **로컬 문맥 의미 검증 (Semantic Penalty Filter)**:
    - API 비용 지출 없이, 타겟 제품 클러스터 유사도를 **3제곱** 연산하는 곱셈 수학 공식을 활용합니다. 엘리베이터, 향수 등의 치트키를 복붙했지만 정작 앞뒤 문맥(자외선 카메라, 블랙박스 등)이 맞지 않는 "아무말 대잔치" 카피는 **점수가 무조건 폭락**하여 걸러지게 하는 완벽무결한 문맥 필터망을 구축했습니다.

## 🚀 운영 가이드
1. 스레드 포스팅 후 성과가 쌓이면 `데이터 참조/dotori.xlsx`를 업데이트합니다.
2. `mab_copy_system_v3.py`를 실행합니다.
3. 시스템이 최신 성공 사례를 학습하여 더 진화된 카피를 추천합니다.
